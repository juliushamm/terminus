What's the primary canvas interaction model?

How you interact with the main workspace — this shapes everything about the UI

Node Graph

Drag-and-drop nodes connected by edges. Like draw.io or Terraform's visual graph. Resources are nodes; relationships are arrows. Freeform spatial arrangement.

Topology / Region Map

Resources nested inside their actual AWS hierarchy — Region → VPC → Subnet → Resource. Mirrors real AWS structure. Spatial nesting shows containment, not just connection.

Hybrid — Both Views

Topology map as the default "architect's view," with a toggle to free-form node graph for relationship exploration. Two lenses on the same infrastructure.
