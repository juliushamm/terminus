How should users create a new resource?

Three interaction patterns for triggering resource creation in M2

A

Palette Drag-and-Drop

User drags a resource type (VPC, EC2, SG) from the left sidebar palette onto the canvas. A "pending" node appears where they drop it, and the right Inspector panel opens with a creation form.

⊞ EC2

drop on canvas
→
Inspector opens with form
B

Create Button in Sidebar

Each resource type in the left sidebar has a "+" button. Clicking it opens a creation form in the Inspector panel (right sidebar). Resource only appears on canvas after the user runs the CLI command and it succeeds.

+ EC2
click →
Inspector: creation form
→
CommandDrawer: aws ec2 run-instances…
C

Canvas Context Menu

Right-click anywhere on the empty canvas (or on an existing node) to get a context menu. "New Resource" submenu lists VPC, EC2, SG. Selection opens the Inspector form. Feels like a design tool (Figma, Miro).
