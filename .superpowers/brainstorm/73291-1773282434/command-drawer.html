Command Drawer — execution states

What should the drawer look like while a command is running and after it completes?

A

Expandable strip — streaming log

Drawer stays as a thin strip at the bottom. On Run, it expands upward to show streaming stdout/stderr. Collapses back to strip on success or dismiss.

collapsed (default)
$ aws ec2 create-vpc --cidr-block 10.0.0.0/16
Run
Cancel
expanded during/after run
$ aws ec2 create-vpc --cidr-block 10.0.0.0/16
{
  "Vpc": {
    "VpcId": "vpc-0abc1234",
    "State": "pending"
✓ Success
✕ dismiss
B

Fixed-height drawer — scrollable log

Drawer is always a fixed taller height (e.g., 120px). Log area is always visible, scrollable. No expand animation — always ready.

$ aws ec2 create-vpc --cidr-block 10.0.0.0/16
Run
Cancel
— output —
$ aws ec2 create-vpc --cidr-block 10.0.0.0/16
{
  "VpcId": "vpc-0abc1234"
  "State": "pending"
}
✓ exit 0
