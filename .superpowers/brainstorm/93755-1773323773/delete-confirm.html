Delete confirmation

Right-click a node → Delete → what happens next?

A

Simple confirm dialog

Small modal: resource name + "This cannot be undone." + Cancel / Delete buttons. Fast, low friction.

Delete vpc-0abc1234?
This action cannot be undone.
Cancel
Delete
B

Type to confirm

Modal with a text input — user must type the resource name or "delete" before the button enables. Matches AWS console UX, adds friction for safety.

Delete vpc-0abc1234?
Type vpc-0abc1234 to confirm
Cancel
Delete
C

Command Drawer confirmation

Same pattern as Create — delete generates an aws ec2 delete-vpc command shown in the drawer. User clicks Run to execute. Consistent with the M2 write model.

$ aws ec2 delete-vpc --vpc-id vpc-0abc1234
Run
Cancel
