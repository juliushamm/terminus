M3 — Where do Edit and Delete live?

Select a node → actions appear somewhere. Where?

A

Inspector panel (right sidebar)

Click a node → Inspector expands with resource details, an Edit button (opens modal, same style as Create), and a Delete button with confirmation. Same place you already read config.

Right-click on the node

Same context menu that triggers Create on the canvas background — right-clicking a node instead shows Edit and Delete for that resource. Canvas right-click still creates.

C

Both — Inspector for Edit, right-click for Delete

Edit opens a full form in the Inspector (in-place, no modal). Delete is a right-click action with confirmation. Separates the two flows: deliberate edits use the sidebar, quick destructive ops use the menu.
